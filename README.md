# Mechilles Webhook
> Discord webhook that delivers messages about Travis CI or Azure Pipelines builds with support for other drivers.

## Requirements
- Node >= 10.15.3
- NPM >= 6.4.1


## Setup
- Get the code
- `npm install`
- `cp .env.example .env`
- Configure the `.env` file
- `npm run build`
- `npm start`
- Good to go!
